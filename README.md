# Currency Conversion Monitor

Converts currency from one to another and monitors the conversion rate.

planned currency pairs:
- [ ] VES (Monitor Dolar) to USD
- [ ] VES (BCV) to USD

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
